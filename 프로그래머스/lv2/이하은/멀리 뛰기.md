## bottom-up DP (반복문)

```java
import java.util.*;

class Solution {
    public long solution(int n) {
        int[] dp = new int[n + 2];
        
        dp[1] = 1;
        dp[2] = 2;
        
        for(int i = 3; i <= n; i++){
            dp[i] = (dp[i - 1] + dp[i - 2]) % 1234567;
        }
        
        return dp[n];
    }

}
```

## top-down DP (재귀 호출)

피보나치 수열과 유사하다! 

```java
import java.util.*;

class Solution {
    private static final int MAX = 2001;
    int[] dp = new int[MAX];
    
    public long solution(int n) {
        return dfs(n);
    }
    
    int dfs(int x) {
        if (x == 1 || x == 2) {
            return x;
        }

        if (dp[x] != 0) {
            return dp[x];
        }

        dp[x] = (dfs(x - 1) + dfs(x - 2)) % 1234567;
        return dp[x];
    }
}
```
